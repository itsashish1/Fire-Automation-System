# Fire Automation System

## Overview
The Fire Automation System is designed to detect and respond to fire emergencies effectively and efficiently.

## Features
- **Real-Time Monitoring**: Constantly monitors the environment for any signs of fire.
- **Automated Response**: Initiates measures such as alarms and notifications immediately upon detection.
- **User-Friendly Interface**: Provides an intuitive interface for users to monitor the status.

## Installation
Follow the steps below to install the Fire Automation System:

1. Clone the repository:
   ```bash
   git clone https://github.com/itsashish1/Fire-Automation-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Fire-Automation-System
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the Fire Automation System, run the following command:
```bash
npm start
```
